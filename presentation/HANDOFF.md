# Dissertation Proposal Defense Deck — Handoff

## What this is
A 28-slide PowerPoint presentation for Spencer Low's PhD dissertation proposal defense at BYU (Computer Science, April 2026). Title: "Bridging Domain Gaps in Visual Correspondence: From Multi-Modal Fusion to Data-Curated Motion Latents."

Planned defense: **August 21, 2026**

---

## Files in this package

| File | Purpose |
|---|---|
| `dissertation_proposal_defense.pptx` | The deck (28 slides, speaker notes on all) |
| `dissertation_proposal_defense.pdf` | PDF export for previewing |
| `build_pptx.js` | Full pptxgenjs source that generates the .pptx |
| `add_notes.py` | Python script that injects speaker notes via python-pptx |
| `render_tables.py` | Python script that renders LaTeX tables as PNG images |

---

## How to rebuild from scratch

```bash
# 1. Install deps
npm install -g pptxgenjs
pip install python-pptx --break-system-packages

# 2. Copy all uploaded figure PNGs/JPGs to /home/claude/
#    (see "Figure manifest" below)

# 3. Build the PPTX
node build_pptx.js

# 4. Inject speaker notes
python add_notes.py

# 5. (Optional) Rebuild table PNGs
pip install matplotlib --break-system-packages
python render_tables.py
```
